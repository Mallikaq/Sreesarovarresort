# Sree Sarovar Resort Image Checklist

Add final photos in this folder and update the HTML placeholders to real `<img>` tags.

Recommended first batch:

- Wide hero photo: front view, pool, garden, lake, or best resort angle
- Exterior entrance photo
- Reception or lobby photo
- Each room type: 2-3 photos per room
- Bathroom photos
- Restaurant and food photos
- Event lawn or banquet setup photos
- Parking and approach road photo
- Nearby attraction photos, if allowed

Best sizes:

- Hero image: 1800 x 1100 or larger
- Room/gallery images: 1200 x 900 or larger
- Keep files under 500 KB each if possible before deployment
